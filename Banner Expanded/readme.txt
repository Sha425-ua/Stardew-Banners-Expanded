# Banners Expanded

Banners Expanded is a decorative mod for Stardew Valley that adds a collection of unique, thematic banners to the game. While most can be purchased at shops or earned during festivals, one exclusive banner is hidden deep within Omni Geodes!

Installation
1. Install the latest version of SMAPI.
2. Install Content Patcher and Stardew Valley Expanded.
3. Extract the [CP] Banners Expanded folder into your Stardew Valley/Mods directory.
4. Launch the game!

Requirements
* Stardew Valley version 1.6 or higher.
* Content Patcher.
* Stardew Valley Expanded (SVE): recommended.
